## 寻星大师加适配器的用法
![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/dongle_cn.png)

### 适配器端
- 将适配器与卫星锅高频头相连，并使用充电宝供电

### 手机端
- 打开蓝牙和GPS功能
- 打开寻星大师, 等待连接图标从 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/unconnect.png) 变成 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/bluetooth.png)，并且不闪烁，表示蓝牙已连接
- 选择卫星转发器，并设置天线相关参数
  > ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/setting_cn.png)
- GPS定位成功后，按照红色箭头 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/arrow.png)所指示的方向来调整卫星锅
- 信号锁定以后，信号强度会变成绿色，微调，使信号强度和信号质量达到最佳状态
- 点击搜索按钮 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/search.png) 搜台，并检查节目是否正确

## 寻星大师加机顶盒的用法
![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/stb_cn.png)

### 机顶盒端
- 首先确保机顶盒已经连接到网络
- 扫描机顶盒上二维码 "Find Satellite"，使手机连接机顶盒
	> ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/boxmenu.png)
### 手机端
- 打开GPS定位功能
- 打开寻星大师, 选择卫星转发器，并设置天线相关参数
  > ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/setting_cn.png)
- 打开寻星大师, 等待连接图标从 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/unconnect.png) 变成 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/net_connected.png)，表示连接成功
- GPS定位成功后，按照红色箭头 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/arrow.png)所指示的方向来调整卫星锅
- 信号锁定以后，信号强度会变成绿色，微调，使信号强度和信号质量达到最佳状态
- 点击搜索按钮 ![image](https://github.com/DVBFinder/DVBFinder/blob/master/pic/search.png) 搜台，并检查节目是否正确
